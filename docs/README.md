# Abhi's Portfolio

A beginner-friendly personal portfolio website built with HTML and CSS only.

## Project Overview

This portfolio showcases my learning journey in web development, featuring:
- Clean, modern design
- Responsive layout (mobile-first)
- CSS animations and transitions
- Semantic HTML structure
- Accessibility considerations

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Variables, Animations)

## Project Structure

```
portfolio/
├── index.html          # Main portfolio page
├── style.css           # Main stylesheet
├── projects/
│   └── profile-card/
│       ├── index.html  # Profile card project
│       └── style.css   # Profile card styles
├── images/
│   └── profile.jpg     # Profile image placeholder
└── README.md           # This file
```

## Features

### Main Portfolio
- **Header**: Logo and navigation
- **Hero Section**: Introduction with CTA buttons
- **About Section**: Learning journey and skills
- **Projects Section**: Showcase of beginner projects
- **Contact Section**: Contact information
- **Footer**: Copyright notice

### Profile Card Project
- Simple profile card with image, name, role, and bio
- Social media links (placeholders)
- Hover animations (lift effect and shadow transition)

## Getting Started

1. Clone or download the project
2. Open `index.html` in your web browser
3. Navigate through the sections using the menu
4. View the profile card project in `projects/profile-card/index.html`

## Customization

- Replace `images/profile.jpg` with your actual profile picture
- Update contact information in `index.html`
- Modify colors and fonts in `style.css` using CSS variables
- Add more projects by following the existing structure

## Browser Support

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive
